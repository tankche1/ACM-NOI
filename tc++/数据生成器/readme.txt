《水管局长》数据生成器使用说明

请先检查目录下是否包含以下文件（缺少任何一个都将无法生成数据）：
tube.exe
gen.exe
run.exe

在此目录下新建一个config.txt文件，此文件为数据配置文件，其中的内容格式如下：
第一行一个整数t，表示测试数据组数。
下面总共有t行，每行5个整数：n、m、q1、q2、flag。
其中n和m的含义如题目所述，q1为操作一的次数，q2为操作二的次数，flag为0或1，1表示此测试点为构造数据，否则为随机数据。
注意：随机数据中，树的直径会非常短，使得朴素算法速度非常快。

范围约定：
n <= 100,000
m <= 1,000,000
q1 + q2 <= 100,000
q2 + n - 1 <= m
m <= n * (n - 1) / 2
flag = 0 or 1

下面为config.txt的默认内容：
10
1000 100000 80000 20000 1
1000 100000 60000 40000 1
10000 500000 80000 20000 0
10000 500000 60000 40000 0
100000 1000000 90000 10000 1
100000 1000000 80000 20000 1
100000 1000000 70000 30000 1
100000 1000000 70000 30000 1
100000 1000000 70000 30000 1
100000 1000000 70000 30000 1

创建完config.txt后，就可以运行run.exe生成数据了，数据生成的时间可能较长，请耐心等待。
